# 📌 3D Digital Cultural Artifact Museum

A web-based application that allows users to interactively explore cultural artifacts in **3D**, including rotation, zoom, and detailed viewing, accompanied by simple explanations.

- Use simple gestures to:
  - Rotate
  - Zoom
  - Explore details
- Read **short and easy explanations** about each artifact

# ⚠️ Risks Identified

### 1. Performance Issues with 3D Models
Large models may cause slow loading or lag.

**Mitigation:**
- Use `.glb` format  
- Compress models  
- Limit polygon count  

---

### 2. User Interaction Complexity
Users may find controls confusing.

**Mitigation:**
- Simplify controls (drag = rotate, scroll = zoom)  
- Add UI hints
